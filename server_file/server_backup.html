AI Impact Tracker Dashboard

AI Impact Tracker estimates the environmental footprint of ChatGPT responses by tracking the number of output tokens in a response. (Tokens are sort of like the word count.) Using the user's coarse location, based on the country, or state or province and watershed (if in the United States or Canada), the extension estimates carbon, water, and energy usage associated with that activity. The extension only collects the number of output tokens and coarse location data. It does not collect or store the contents of user prompts or responses. Usage data is sent to aiimpacttracker.cs.haverford.edu so users can view total carbon, water, and energy usage across all users.
